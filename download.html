Download OlyProbe

OlyProbe is an app for Windows, Mac, and Linux systems that enables you to create an unlimited number of custom "Setting Cheats" for instant, exact camera setup when needed. It supports a wide range of Olympus/OM SYSTEM camera models. See for details.

OlyProbe is a simple application for reading and writing configuration information with your camera and saving, using, and uploading the .cheat files you create. It consumes almost no system resources, doesn't require any drivers, and has little interaction with your operating system.

This is beta software and does not yet have the certificate needed to avoid security warnings during installation. You can power through those warnings and install it safely — see the security note below. The production version will not have this restriction.

Supported Cameras

OlyProbe supports all Olympus and OM SYSTEM cameras that expose the OPC WiFi API. USB tethering is supported on pro-tier bodies only, in line with OM SYSTEM's own tethering support policy.

About the security warning

Code signing certificates for desktop software cost several hundred dollars per year and are issued by certificate authorities that verify the identity of the developer. As beta software from a small independent lab, OlyProbe is not yet signed. This means Windows will flag it with a SmartScreen warning and macOS will require a manual override in System Settings. These warnings are about certificate status, not actual risk. On Windows, click "More info" and then "Run anyway." On macOS, go to System Settings → Privacy & Security and click "Open Anyway" after the first blocked launch attempt. OlyProbe does not connect to the internet during camera operations — it communicates only with your locally connected camera over USB or your local WiFi network. The production release will be properly signed and will not trigger these warnings. We are happy to answer any questions at olyprobe@outlook.com.
